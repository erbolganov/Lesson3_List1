2)  Найти сумму элементов массива с четными номерами, содержащего N элементов. Элементы вводятся с клавиатуры.
3) Найти наименьший элемент одномерного массива, состоящего из n элементов. Элементы вводятся с клавиатуры.
4)  В заданном одномерном  массиве, состоящем из n  целых чисел, подсчитать количество нулей.
5) В заданном одномерном  массиве, состоящем из n  целых чисел, подсчитать количество четных элементов.
6) Массив А вводится с клавиатуры. Сформировать новый массив В, состоящий из четных элементов массива А. Элементы вводятся с клавиатуры. Размер n.
7) Введите с клавиатуры в массив пять целочисленных значений. Выведите их в одну строку через запятую. Получите для массива среднее арифметическое.
8) Создайте массив из 15 целочисленных элементов и определите среди них минимальное значение.
9) Найти количество четных элементов одномерного массива.
10) Вычислить сумму четных элементов одномерного массива до первого встреченного нулевого элемента.
